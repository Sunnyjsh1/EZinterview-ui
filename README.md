# 이지인터뷰 UI 프로토타입

**모바일 AI 모더레이터 서비스 와이어프레임**

버전: 1.0  
제작일: 2026년 1월  
형식: HTML/CSS 인터랙티브 프로토타입

---

## 📋 프로젝트 개요

이지인터뷰는 AI 기반 모바일 인터뷰 서비스로, 관리자와 사용자를 위한 완전한 조사 프로세스를 제공합니다. 본 프로토타입은 서비스의 전체 화면 구조를 와이어프레임 형태로 구현한 HTML 기반 프로토타입입니다.

### 주요 특징
- ✅ **총 22개 화면** 완전 구현
- ✅ **와이어프레임 스타일** 디자인
- ✅ **인터랙티브 네비게이션** (이전/다음 버튼)
- ✅ **반응형 레이아웃** (모바일/데스크톱)
- ✅ **Figma 변환 최적화** 구조
- ✅ **인쇄 친화적** 레이아웃
- ✅ **PPT 변환 가능** 디자인

## 🚀 사용 방법

### 1. 프로토타입 실행
```bash
# 브라우저로 index.html 파일 열기
open index.html
# 또는
start index.html
```

### 2. 화면 탐색
- **목차 페이지**: 전체 화면 목록 확인
- **각 화면**: 클릭하여 상세 보기
- **네비게이션**: 하단 이전/다음 버튼으로 이동
- **목차 복귀**: 중간의 "목차" 링크 클릭

### 3. Figma로 변환
1. Figma 플러그인 설치: **"HTML to Figma"**
2. 플러그인 실행
3. HTML 파일 또는 URL 입력
4. 변환 실행

### 4. PDF/이미지로 출력
```
브라우저 인쇄 기능 사용 (Ctrl+P / Cmd+P)
- 대상: PDF로 저장
- 레이아웃: 세로
- 여백: 기본값
```

### 5. PPT 변환
1. 각 화면을 브라우저에서 스크린샷 촬영
2. PowerPoint에 이미지로 삽입
3. 또는 PDF로 저장 후 PPT에 import

## 💡 주요 기능

### 관리자 기능
1. **프로젝트 등록**: NKMS 번호 입력, 조사 정보 관리
2. **가이드 검토**: 사용자 생성 가이드 승인/수정 요청
3. **테스트 링크 발송**: 응답자에게 인터뷰 링크 전달
4. **실사현황 관리**: 실시간 응답 현황 모니터링
5. **응답내용 검수**: 녹음 파일 및 스크립트 품질 검수

### 사용자 기능
1. **모의 견적**: 실시간 비용 산출 및 조사 문의
2. **약정서 승인**: 전자서명 기반 계약 체결
3. **가이드 생성**: AI 추천 기반 질문 구성
4. **분석결과 확인**: 스크립트, 인사이트, 리포트 다운로드
